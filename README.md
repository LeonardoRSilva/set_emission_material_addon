## Set Emission Material (Blender Addon)

### Scripts Blender
- **Set Emission Material**: When importing a <code>.svg</code> file into Blender, objects by default have a material type Diffuse BSDF  that doesn't use nodes, when trying to change to node Emission, the defined previously color is lost. This addon replaces the default material with an emission node, maintaining the previously assigned color of the material.

### How install(manual):

- Download .zip file.
- Go to Blender menu <code>Edit/Preferences/addons</code>.
- Select options <code>Install from disk</code> and select .zip downloaded.

 ![install](https://github.com/LeonardoRSilva/set_emission_material_addon/blob/main/imgs/install.png)

### How to use:

- In 3dView, press N in keyboard and select <code>Emission</code> tab.
- Press button <code>Set Emission Material</code>.

![addon](https://github.com/LeonardoRSilva/set_emission_material_addon/blob/main/imgs/addon.png)
 
All materials will be changed to <code>Emission</code> type while maintaining the color of the object.
